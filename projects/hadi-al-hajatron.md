# Hadi Al Hajatron

Repository: https://github.com/mohammednafees1007-hub/Hadi-Al-Hajatron

## Summary

Hadi Al Hajatron is a robotics project combining Arduino firmware, Python desktop tooling, live map building, obstacle sensing, and A* path planning.

## What Was Built

- Live grid-mapping GUI in Python/Tkinter.
- Arduino motor and encoder firmware.
- IR-based obstacle sensing.
- Frontier exploration logic.
- Saved map output for planner handoff.
- A* path planner and command generation.

## Resume Bullet

Built a 4x4 Arduino-Python autonomous grid-navigation robot with 3 IR obstacle sensors, 60 cm calibrated movement steps, live Tkinter mapping, JSON map export, and A* route planning.
