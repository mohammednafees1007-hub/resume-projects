# AI Image Analyzer

Repository: https://github.com/mohammednafees1007-hub/ai-image-analyzer

## Summary

AI Image Analyzer is the backend image-analysis module for the larger Visage project. The repo focuses on backend computer vision output, not the full Visage frontend or product workflow.

## What Was Built

- PIL image loading.
- YOLOv8n object detection through Ultralytics.
- Allowed-class filtering.
- Label remapping for app-friendly object names.
- OpenCV Haar cascade face detection.
- Privacy-safe face tokens such as `Face_1`.
- BLIP scene captioning through Transformers.
- JSON aggregation through `run.py`.

## Resume Bullet

Built the backend image-analysis pipeline for a larger Visage project, combining object detection, privacy-safe face tokens, BLIP scene captions, and clean JSON output for app integration.
